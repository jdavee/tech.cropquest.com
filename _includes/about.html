About

Here are some of our milestones.

  • 2011

    Tech Team Division

    Crop Quest's IT and Precision Ag teams coalesced into a single division to support, develop, and implement company wide technology and precision agriculture initiatives.

  • 2011-2015

    Internal App Development and Modernization

    During this time period we mainly focused on buildng (or re-building) modern apps to handle various internal company processes like expense entry/tracking, fleet management, contracts, etc. We also virtualized and expanded our server infrastructure to support these new tools.

  • 2015

    Ag Dispatching Launches

    Ag Dispatching is a logistcs and dispatching tool used internally by our Precision Ag team, but it is built to support use by external organizations. It uses Ruby on Rails, Sidekiq, PostgreSQL, PostGIS, and Bootstrap.

  • 2016

    Water Meter Log Launches

    Water Meter Log is an irrigation meter recording, sharing and reporting app. It was the first Software as a Service web application we launched. It uses Ruby on Rails, PostgreSQL, and jQuery Mobile.

  • 2017-2018

    Paramount Reports Launches

    Paramount Reports is a tool for building professional customizable reports for agX compliant software including SST Software's Summit and Sirrus. This was a complete web-based rewrite of our original desktop application. It uses Ruby on Rails, Sidekiq, PostgreSQL, PostGIS, and Bootstrap.

  • 2017-2018

    Satellite Imagery Processing (Zodiac)

    Zodiac is a satellite imagery processing platform we created to deliver NDVI images through Paramount Reports and other agX compliant software like SST Sirrus. It is currently built with Ruby and Python.
