Technology

These are some of the main technologies we work with:

Ruby on Rails

Ruby on Rails is the predominant framework we use to build our web apps and APIs.

PostgreSQL

PostgreSQL is used for all of our web apps, commonly in conjunction with PostGIS.

Swift

We are beginning to build iOS apps to consume our web application APIs with Swift.

Linux

All of our web applications are deployed to Linux servers that we manage.

GitHub

All of our projects are hosted in GitHub repositories. We use forking, feature branches, and pull requests extensively during the development process.

Trello

We use trello to manage all of our project tasks.
